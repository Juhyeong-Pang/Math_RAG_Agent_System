# RAG Agent System for AIMO3

## Description

The core motivation for this project is to implement an RAG Agent System that overcomes LLM's limits.

I chose [AIMO3 Kaggle Competition](https://www.kaggle.com/competitions/ai-mathematical-olympiad-progress-prize-3/overview) to see how RAG system can enhance LLM model's performance when solving math questions.

- Relatively smaller LLM models such as GPT-o4 is performing terrible at solving such math problems (GPT-o4 correctly solved 0 question from the AIMO3_Reference_Problems).

- This meant that the improvement of the model's performance shown in this project will solely be due to the implementation of RAG agent system.
